Política de privacidade

Leia atentamente os termos de uso, só use nosso aplicativo se realmente estiver de acordo com o descrito abaixo, obrigado pela sua visita.

política de privacidade para cbersoftware

Ao utilizar nossos aplicativos, os usuários nos confiam algumas informações. 1. Coleta de Dados

Nosso aplicativo coleta e utiliza informações pessoais dos usuários conforme descrito abaixo:

Dados Pessoais: Coletamos informações como nome, endereço de e-mail e outras informações de contato quando o usuário se registra ou interage com nosso aplicativo. Dados de Uso: Registramos dados sobre a forma como o usuário utiliza o aplicativo, incluindo páginas visitadas, recursos acessados e tempo gasto no aplicativo.

2. Uso dos Dados Utilizamos os dados coletados para os seguintes fins:

Melhoria do Aplicativo: Analisamos os dados para aprimorar a experiência do usuário, corrigir erros e otimizar o desempenho. Comunicação com o Usuário: Enviaremos notificações relevantes e informações sobre atualizações do aplicativo.

3. Compartilhamento de Dados Não compartilhamos informações pessoais com terceiros, exceto quando necessário para cumprir obrigações legais ou melhorar o aplicativo.

4. Segurança Tomamos medidas para proteger os dados dos usuários contra acesso não autorizado ou uso indevido.

5. Consentimento Ao usar nosso aplicativo, o usuário concorda com esta política de privacidade.
